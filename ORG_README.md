## 👋 Who we are

MoonRune is an independent open-source organization building software with a simple philosophy: **tools should work for the people who use them**, not against them.

We care about transparency, security, and giving communities real control over their own spaces and data. We build for the long term — no dark patterns, no opaque systems, no lock-in.

## 🚀 Projects

### 🌙 [Cauldron](https://giteas.fullmooncyberworks.com/MoonRune/Cauldron)

> *A FOSS Discord alternative that respects your community.*

A free, open-source chat platform built around transparency, security, and community control. Create servers, organize channels, and chat in real time — with a clean modern interface and none of Discord's frustrations.

**Highlights:**
- 🔒 Security-first by design — replay attack detection, 2FA, encrypted secrets
- 🏘️ Community-controlled servers with transparent role structure and invite management
- 🔑 Server succession — communities don't die because an account gets locked
- 🖥️ Native desktop (Windows, macOS, Linux), Android, and web — one codebase via Tauri
- 🐳 Self-hostable with a single `docker compose up`

**Deploying at:** [`chat.moonrune.cc`](https://chat.moonrune.cc)

### 🤖 Mei

> *An AI companion with genuine presence.*

Mei is a personal AI companion built for depth over novelty. She has a consistent character, a memory that evolves across conversations, emotional awareness, and voice support — designed to feel like a real presence rather than a chatbot.

**Highlights:**
- 🧠 Persistent memory that summarizes and evolves over time
- 💬 Consistent, grounded personality — dry, understated, not theatrical
- 🎙️ Voice interface (STT + TTS) with latency targets that feel natural
- 🔗 Integrates with Discord and Telegram
- 🏠 Runs locally — your data stays yours

---

### 🏕️ PocketColony

> *A colony sim that fits in your pocket.*

PocketColony is a mobile-first colony simulation game built in Godot 4. Manage colonists, gather resources, construct buildings, and survive events — in Sandbox or Survival mode. Designed for short sessions and deep replayability.

**Highlights:**
- 🎮 Godot 4 + GDScript — fully open source
- 👥 Colonist AI with role assignment, needs, and personality traits
- 🌲 Resource system, world grid, AStar pathfinding, and dynamic events
- 📱 Android-first, built for touch input
- 💾 Save system with multi-slot backup rotation

## 🤝 Contributing

All MoonRune projects are open source and welcome contributions. Check the `README.md` and open issues in each project repo to find a good starting point.

We value:
- **Clear, well-scoped contributions** — small and focused over large and sweeping
- **Security awareness** — flag anything that looks wrong, even if it's not your area
- **Respect for the existing architecture** — read the docs before proposing structural changes
